# 这是什么

作为一个机器人爱好者，在探索途中遇到的各种问题，有迷茫，有自卑，但是我从不放弃。如果你在前进途中也有类似想法，不要沮丧，努力前行。星光不问赶路人，时光不负有心人。

互联网时代，碎片化的知识太多，但是大多都不成体系，关联性较小。机器人人相关的知识更是如此，大部分的知识都缺少上下文，不利于学习和理解。所以我整理了个人在机器人方面的点滴知识。
